# 🧱 SafePlay

SafePlay is a browser-based voxel sandbox game inspired by Minecraft and built with Three.js.

The project is focused on learning game development, 3D graphics, world generation, and voxel engine mechanics from scratch.

## ✨ Features

- 🎮 First-person camera controls
- 🖱️ Mouse look with pointer lock
- ⛏️ Break blocks
- 🧱 Place blocks
- 🎯 Center crosshair
- 💡 Dynamic lighting
- 🌍 16×16 voxel world
- 🚀 Smooth movement controls

## 🎮 Controls

| Key | Action |
|------|---------|
| W | Move Forward |
| A | Move Left |
| S | Move Backward |
| D | Move Right |
| Mouse | Look Around |
| Left Click | Break Block |
| Right Click | Place Block |
| Space | Fly Up |
| Shift | Fly Down |

## 🚀 Running Locally

1. Clone the repository

```bash
git clone https://github.com/your-username/safeplay.git
```

2. Open the project folder

```bash
cd safeplay
```

3. Run a local server

Example using VS Code Live Server:

- Install the Live Server extension
- Right-click `index.html`
- Click **Open with Live Server**
